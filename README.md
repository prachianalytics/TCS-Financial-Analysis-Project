# TCS Financial Performance Analysis (FY2021 - FY2025)

## 📌 Project Overview
A comprehensive five-year empirical study of Tata Consultancy Services Ltd (TCS). This analysis evaluates revenue growth, profitability, and solvency using advanced Excel techniques and statistical forecasting.

## 🛠️ Tools & Skills
* **Excel:** Ratio Analysis, Data Cleaning, Pivot Tables.
* **Statistical Analysis:** Linear Regression, Correlation Matrix (Data Analysis ToolPak).
* **Financial Modeling:** 5-year CAGR and margin analysis.

## 📁 Repository Structure
* **`/Data`**: Raw financial data and calculation sheets.
* **`/Documentation`**: Full Word report and PPT presentation.
* **`Final_Report.pdf`**: Merged project document for quick review.

## 💡 Key Insights
* **Revenue:** Steady growth reaching ₹2,55,324 Cr in FY2025.
* **Stability:** Maintained a healthy Net Margin (19%) and low Debt-to-Equity ratio.
